Belly Button Biodiversity Dashboard

Use the interactive charts below to explore the dataset

This page was created to fulfill challenge 12 of the UC Berkeley Data Analytics and Visualization Bootcamp. This interactive page allows users to explore a dataset of skin microbiome sequences from anomalized patients. Data was provided in json format as part of the course materials. Select a patient number from the dropdown menu to see demographic information, the top then bacterial species identified, and sample diversity. Mouse over bar or bubble graph data to see identified species names (note: not all OTUs have full species names).
